# Endless Runner — 驗收測試文件 (QA)

## 使用說明

- 每個 Step 完成後，DEV 會開 Pull Request 並附上 Web build 網址
- QA 用瀏覽器開啟網址，依據該 Step 的測試項目逐項驗證
- 全部通過打勾後，在 PR 留言「QA Pass」
- 有問題在 PR 留言描述問題，附截圖或錄影

## 測試環境

| Step | 測試方式 |
|---|---|
| 2 - 9 | 瀏覽器開啟 Web build 網址（Chrome / Firefox） |
| 10 | Android 手機安裝 APK |
| 11 | Windows PC 執行 .exe |
| 12 | iPhone 安裝測試版 |
