# Endless Runner — QA 測試報告

## 使用說明

1. DEV 完成一個 Step 後，會提供一個**網址**給你
2. 用瀏覽器（Chrome 或 Firefox）打開該網址
3. 找到對應 Step 的表格，照著「測試步驟」欄位操作
4. 在「結果」欄位填入 **Pass** 或 **Fail**
5. 如果是 Fail，在「備註/截圖」欄位貼上截圖或描述看到的狀況
6. 全部測完後，把結果回覆給 PM

## 名詞說明

| 名詞 | 意思 |
|---|---|
| Web build | 開發人員打包好的網頁版遊戲，用瀏覽器打開就能玩 |
| Console | 瀏覽器的開發者工具，按 F12 可開啟，切到 Console 分頁看錯誤訊息 |
| 重新整理 | 瀏覽器按 F5 或點擊重新載入按鈕 |

## 回報問題的方式

測試時如果遇到 Fail 的項目，請在「備註/截圖」欄位提供以下資訊：

- **看到了什麼**：簡短描述你看到的狀況（例如：「角色穿過地板掉下去了」）
- **截圖或錄影**：用手機拍螢幕或用截圖工具擷取畫面
- **發生時機**：大約是遊戲開始後幾秒發生的（例如：「開始後大約 15 秒」）
